### 安装依赖

git clone 项目到本地，直接用 HBuilderX 打开，直接运行即可，不需要安装任何依赖


### 启动应用

```bash
# yarn 
yarn serve

# npm 
npm run serve

# pnpm 
pnpm serve
```

## 发布

```bash
npm run build
```

```bash
npm run preview
```
